# Akan-Name-Generator
The naming is more than just a name in this culture, it's connecting to each individual through names.

# Project Description
 It's a Ghanaian culture system built, where children are named based on the day they were born and also depends on the childs gender.

# Author 
Erick Ochieng

# Setup Instructions
1. Cloned the repository :

2. Opened the repository from Ubuntu to VS code

3. Access files on the VS code

4. Open the index.html on your browser.

# BDD(Behavior-Driven Development)
- Enter valid day
- Enter valid month
- Enter valid year
- Enter gender 
- Press the know your akan name 
- Generates your akan name
- The screen will reset after 15 secs of idling
